# Titanic Data Analysis Project 🚢

## 📌 Overview
This project is a data analysis of the Titanic dataset. The goal is to understand the factors affecting passenger survival using data cleaning, statistical analysis, and visualization techniques.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
The dataset used in this project is the Titanic dataset, which contains information about passengers such as age, gender, ticket class, fare, and survival status.

---

## ⚙️ Data Cleaning
- Filled missing values in **Age** using mean
- Filled missing values in **Embarked** using mode
- Dropped **Cabin** column due to excessive missing values

---

## 📊 Analysis Performed
- Survival count analysis
- Gender-based survival analysis
- Passenger class analysis
- Age distribution analysis
- Correlation analysis

---

## 📈 Visualizations
- Count plots
- Histograms
- Heatmaps
- Boxplots

---

## 🔍 Key Insights
- Females had a higher survival rate than males
- First-class passengers had better survival chances
- Higher fare passengers were more likely to survive
- Most passengers were young adults

---

## 📁 Project Files
- `train.csv` – Original dataset
- `cleaned_titanic.csv` – Cleaned dataset
- `Untitled.ipynb` – Jupyter Notebook with full analysis

---

## 🎯 Conclusion
This project demonstrates how data analysis techniques can be used to extract meaningful insights from real-world datasets.
